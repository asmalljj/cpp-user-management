# C++ User Management System (CLI)

这是一个基于 C++ 的命令行用户管理系统，用于课程小组作业。  
系统实现了用户注册、登录、用户档案编辑以及基于 JSON 文件的本地数据持久化，并支持管理员与普通用户权限分离。

---

## 功能概述

### 普通用户功能
- 用户注册
- 用户登录
- 修改昵称（nickname）
- 修改学习信息：
  - 学习目标（goal）
  - 学习地点（location）
  - 学习时间（time）
- 修改个人密码
- 用户数据持久化保存（程序重启后数据不丢失）

### 管理员功能
- 管理员登录（通过管理员口令）
- 删除指定用户
- 重置指定用户的密码
- 管理员与普通用户功能严格分离

---

## 本地数据持久化说明

- 用户数据采用 **JSON Lines（JSONL）** 格式存储在 `data/users.jsonl`
- 每行表示一个用户对象
- 支持：
  - 新用户追加写入
  - 用户信息修改时重写文件
  - 删除用户时重写文件
- 程序重启后仍可正确加载和使用已有数据

## 编译与运行

### 环境要求
- Windows
- MinGW-w64
- g++（支持 C++17）

### 编译

在项目根目录执行：

```bash
g++ -std=c++17 -O2 -I. main.cpp storage/storage.cpp auth/auth.cpp -o main.exe

运行
./main.exe

管理员说明

管理员登录需要输入管理员口令（示例中写死在代码中）

管理员功能包括：

删除用户

重置用户密码

普通用户无法访问管理员功能

说明

本项目为课程作业用途，密码加密采用简单哈希方式，仅用于演示

未引入数据库，采用文件方式实现本地持久化

系统设计清晰，模块划分明确，便于后续扩展
